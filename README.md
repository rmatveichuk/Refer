# Refer - Asset Manager for CGI Professionals

**Refer** — это профессиональный локальный инструмент для управления референсами и визуального поиска, разработанный специально для специалистов в области архитектурной визуализации и CGI.

## 🎯 Основные цели
1.  **От поиска к системе**: Переход от бесконечного скроллинга к структурированной локальной базе знаний.
2.  **Визуальный интеллект**: Использование нейросети SigLIP для поиска по смыслу и визуальному сходству.
3.  **Автономность**: Все данные и нейросети работают локально. Никаких подписок или облачных API.
4.  **Скорость**: Мгновенный доступ к курируемому контенту без отвлечения на посторонний шум.

## 🧠 AI Возможности
-   **SigLIP Search**: Продвинутый поиск по текстовому описанию (например, *"moody sunset lighting architectural concrete"*).
-   **Image-to-Image**: Найдите похожие референсы, просто перетащив (Drag & Drop) любую картинку в окно программы.
-   **Auto-Translate**: Ищите на русском языке — система автоматически переведет запрос для нейросети.

## 🌐 Источники
-   **Behance**: Основной источник данных. Поддерживается парсинг конкретных проектов и целых профилей авторов.

## 🚀 Стек технологий
-   **Python 3.10+ / PyQt6**: Современный десктопный интерфейс.
-   **SigLIP (google/siglip-so400m)**: Государство-арт нейросеть для понимания изображений.
-   **FAISS**: Высокопроизводительная база данных для мгновенного векторного поиска.
-   **SQLite3 (WAL mode)**: Надежное хранилище метаданных.
-   **curl_cffi**: Быстрый и эффективный сбор данных без использования тяжелых браузеров.

## 📥 Установка
1. Клонируйте репозиторий.
2. Создайте виртуальное окружение: `python -m venv .venv`.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Запустите: `python main.py`.
